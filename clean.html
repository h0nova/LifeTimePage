Як організувати прибирання

"Якщо ображають, не ображайся, якщо вдарили, то не вдарався"
Джейсон Стетхем

1. Складіть план Почніть з визначення зон для прибирання:

  • Кухня — приберіть поверхні, вимийте посуд.
  • Вітальня — розкладіть речі по місцях, пропилососьте.
  • Ванна кімната — очистіть дзеркала, продезінфікуйте сантехніку.

3. Використовуйте правильні засоби Для кожної зони підберіть відповідні засоби:

  • Для скла — спреї для очищення скла.
  • Для підлоги — миючі засоби для різних покриттів.

2. Інструменти для прибирання Підготуйте все необхідне:
  • Ганчірки та губки.
  • Пилосос або швабру.
  • Сміттєві пакети.
